Vocabulary

  • Assignment is a reference
  • Variables are used for assignment
  • Functions perform operations

Vocabulary Cont.

  • Args allow functions reuse
  • Objects have properties/attributes
  • Invocation is calling/running a function
  • Methods are function properties of an object

Syntax

  • Parens invoke functions and surround arguments
  • Curlies surround function bodies/blocks
  • Semicolons terminate a statement/expression
  • Indentation indicates nested statements

Syntax cont.

  • Newlines separate statements
  • Double newlines separate unrelated blocks
  • Period delimits object property hierarchy

Types & Native Methods

  • Numbers are integers
  • Functions perform operations and accept arguments
  • Objects are nouns that store properties
  • Strings are strings of text

Types cont.

  • Booleans are true/false
  • Arrays are lists
  • Conditionals
